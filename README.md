# 工作日志与总结

本地运行的加密工作日志应用：记录日程、日志、会议与每日/每周总结，数据经 AES-GCM 加密后保存在浏览器 `localStorage`。

## 快速开始

1. 双击运行 `start-server.bat`（需已安装 Python）
2. 浏览器打开 [http://localhost:8080](http://localhost:8080)
3. 首次使用设置主密码；**请勿用 file:// 直接打开**，否则加密不可用

> 忘记主密码无法恢复数据，请定期在「设置 → 导出加密备份」保存备份文件。

## 功能概览

- 时间线、日历、日志与固定日程
- 每日 / 每周 Markdown 总结
- 主密码 + PBKDF2 + AES-GCM 本地加密
- 加密 JSON 备份导入 / 导出

## 数据备份与恢复

- **导出**：应用内 **设置 → 导出加密备份**，将 `.json` 保存到安全位置
- **导入**：**设置 → 导入加密备份**，选择备份文件并用主密码解锁
- **换电脑**：在新电脑上获取项目文件后，按「快速开始」运行，再执行导入即可

## 技术栈

纯前端：HTML / CSS / JavaScript，Web Crypto API，无后端依赖。
